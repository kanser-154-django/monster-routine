🔥 Real-Time Sync — Firebase (Free)
Sync data instantly between your phone and laptop. Changes on one device appear on the other in seconds. Free forever.
ONE-TIME SETUP (5 min)
1
Go to console.firebase.google.com → sign in with Google
2
Click Add project → name it "monster-routine" → Continue → Create project
3
Click Web icon </> → register app → copy the firebaseConfig object
4
In left sidebar → Build → Firestore Database → Create database → Start in test mode → Enable
5
Paste your config below → Save → open on both devices → they sync instantly ✓

HOW TO SYNC PHONE ↔ LAPTOP
1. On your laptop — click Download Backup → saves monster_backup.json
2. Send that file to your phone (WhatsApp, email, Google Drive — anything)
3. On your phone — open Monster Routine → ⚙️ → Choose File → pick the JSON
4. Done — all data is now identical on both devices ✓
